# Example Proxy Formats for Proxy Checker

# 1. Basic format (IP:Port) - Checker will try HTTPS first, then SOCKS5
12.34.56.78:8080
1.2.3.4:3128

# 2. HTTP/HTTPS Protocol explicitly specified
http://11.22.33.44:80
https://22.33.44.55:443

# 3. SOCKS Protocol (supports socks, socks4, socks5)
socks4://33.44.55.66:1080
socks5://44.55.66.77:1080
socks://55.66.77.88:1080

# 4. Proxies with Authentication (In Development)
# user:pass@ip:port
# admin:secret123@123.123.123.123:8080
